# Demonstration Flow

1. Log in as an Admin user and allocate funds to a beneficiary under a specific category.
2. Switch to the Beneficiary view and attempt a transaction with a merchant.
3. Observe system behavior when category rules are violated and when they are followed correctly.
4. Access the ledger view to verify that all transactions are recorded transparently.
5. Switch to the Donor view to observe aggregated fund utilization and transparency metrics.
